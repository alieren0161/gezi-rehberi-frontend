🎯 1. Projenin Amacı

Bu proje kapsamında dünyanın farklı şehirleri için rehber içerikleri toplanmış, bu veriler Yapay Zekâ (YZ) araçları ile zenginleştirilerek (metin, görsel vb.) çok dilli bir veritabanı kuran ve sonuçları modern bir web arayüzü ile son kullanıcıya sunan uçtan uca bir sistem inşa edilmiştir.

Proje üç ana ayaktan oluşmaktadır: Backend (Strapi CMS), Otomasyon Motoru (Python) ve Frontend (Streamlit).

4.2. Otomasyon Motoru (Python) — automation_engine.py

Aşağıdaki Python betiği, veri toplama, YZ ile zenginleştirme, çeviri, görsel üretimi ve Strapi'ye kayıt sürecini tek tuşla gerçekleştirir:

Ana Fonksiyonlar:

  • fetch_travel_data(): Gezi mekan verilerini havuzdan çeker
  • enrich_and_translate_text(): Metni YZ ile zenginleştirir, deep-translator ile TR→EN çeviri yapar
  • generate_and_download_image(): Pollinations AI API ile mekan görseli üretir
  • upload_image_to_strapi(): Görseli Strapi Media Library'ye (Upload API) yükler
  • save_data_to_strapi(): Tüm verileri JWT Token ile Strapi REST API'ye kaydeder
import requests import time from deep_translator import GoogleTranslator # --- YAPILANDIRMA --- STRAPI_URL = "https://gezi-rehberi-backend-9nmq.onrender.com" STRAPI_TOKEN = "***API_TOKEN***" HEADERS = { "Authorization": f"Bearer {STRAPI_TOKEN}" } def fetch_travel_data(): # Gezi verileri havuzdan çekiliyor return [ {"city_name": "Istanbul", "city_country": "Turkiye", "city_info": "Asya ve Avrupa'yı birleştiren tarihi metropol.", "place_name": "Ayasofya Camii", "base_desc": "Tarihi yarımadada bulunan, dünya mimarlık tarihinin en önemli anıtlarından biri."}, {"city_name": "Tokyo", "city_country": "Japonya", "city_info": "Geleneksel kültür ile ultra modern yaşamın harmanlandığı başkent.", "place_name": "Shibuya Kavsagi", "base_desc": "Dünyanın en yoğun yaya trafiğine sahip, neon ışıklı devasa kavşak."} ] def enrich_and_translate_text(base_desc, place_name): enriched_tr = f"{base_desc} Bu eşsiz mekan, her yıl milyonlarca turisti ağırlamaktadır." enriched_en = GoogleTranslator(source='tr', target='en').translate(enriched_tr) return enriched_tr, enriched_en def generate_and_download_image(place_name): prompt = f"professional travel photography of {place_name}, scenic" url = f"https://image.pollinations.ai/p/{requests.utils.quote(prompt)}?width=1024&height=768" response = requests.get(url, timeout=20) return response.content if response.status_code == 200 else None def upload_image_to_strapi(img_content, file_name): files = {'files': (file_name, img_content, 'image/jpeg')} res = requests.post(f"{STRAPI_URL}/api/upload", headers=HEADERS, files=files) return res.json()[0]['id'] if res.status_code in [200, 201] else None def save_data_to_strapi(city_data, tr_desc, en_desc, media_id): # Şehir oluştur/bul, mekanı TR ve EN olarak kaydet city_payload = {"data": {"Ad": city_data["city_name"], "Ulke": city_data["city_country"]}} # ... Strapi REST API ile kayıt (JWT Auth) def main(): raw_data = fetch_travel_data() for item in raw_data: tr_desc, en_desc = enrich_and_translate_text(item["base_desc"], item["place_name"]) img = generate_and_download_image(item["place_name"]) if img: media_id = upload_image_to_strapi(img, f"{item['place_name']}.jpg") save_data_to_strapi(item, tr_desc, en_desc, media_id)

4.3. Kullanıcı Arayüzü (Streamlit)

Streamlit kullanılarak oluşturulan frontend arayüzü şu özelliklere sahiptir:

  • Strapi API'den GET request ile veri çekme
  • Şehir filtreleme (dropdown ile seçim)
  • Mekan kartlarında görsel, açıklama ve puan gösterimi
  • Çok dilli arayüz — Türkçe ve İngilizce geçiş
  • Modern, dark theme tasarım
  • Responsive layout
